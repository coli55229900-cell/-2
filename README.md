# Camera App — Лабораторні роботи №6 та №7

## Опис
C++ програма для обробки відео з веб-камери в реальному часі.
Використовує OpenCV для захоплення відео, застосування фільтрів
та детекції облич через нейромережу ResNet-10.

---

## Вимоги до системи
- **ОС**: Ubuntu 20.04 / 22.04 (або інший Debian-based Linux)
- **Залізо**: веб-камера, мінімум 2 ГБ RAM
- **ПЗ**: cmake, g++, make, libopencv-dev, wget

---

## Архітектура — опис класів

| Клас | Файли | Відповідає за |
|------|-------|---------------|
| CameraProvider | CameraProvider.hpp / .cpp | Захоплення кадрів з веб-камери через cv::VideoCapture |
| KeyProcessor | KeyProcessor.hpp / .cpp | Обробка натискань клавіатури, зберігання поточного режиму (enum Mode) |
| FrameProcessor | FrameProcessor.hpp / .cpp | Застосування фільтрів до кадру залежно від режиму, виведення статистики |
| MouseHandler | MouseHandler.hpp / .cpp | Обробка подій миші: малювання ліній лівою кнопкою, зум колесом |
| Display | Display.hpp / .cpp | Відображення результату через cv::imshow, створення вікна |
| FaceDetector | FaceDetector.hpp / .cpp | Детекція облич у фоновому потоці через нейромережу ResNet-10 |

## Багатопотоковість (Лаб. №7)

Детекція облич винесена в окремий фоновий потік:

- **Головний потік (UI)**: читає кадри з камери, застосовує фільтри,
  відображає відео. Працює на 30+ FPS без затримок.

- **Фоновий потік (Worker)**: отримує копію кадру, запускає інференс
  нейромережі (~50-200 мс), зберігає координати облич.

- **Синхронізація**: спільні дані (кадр і координати облич) захищені
  std::mutex. Головний потік малює останні відомі рамки не чекаючи
  завершення інференсу — відео залишається плавним.

## Керування

### Клавіші — режими обробки
| Клавіша | Режим |
|---------|-------|
| 1 | Normal — без фільтрів |
| 2 | Invert — інверсія кольорів |
| 3 | Blur — розмиття (Gaussian Blur) |
| 4 | Canny — виділення країв (Canny filter) |
| 5 | Sobel — фільтр Собеля |
| 6 | Binary — бінаризація |
| 7 | Glitch — ефект гліча (зсув RGB каналів) |
| 8 | Picture-in-picture — накладання картинки |
| F | Face Detection — детекція облич (нейромережа) |
| ESC | Вийти з програми |

### Клавіші — керування
| Клавіша | Дія |
|---------|-----|
| Стрілка вгору / вниз | Переміщення хрестика вертикально / поворот кадру |
| Стрілка вліво / вправо | Переміщення хрестика горизонтально |
| + | Збільшити яскравість |
| - | Зменшити яскравість |

### Миша
| Дія | Результат |
|-----|-----------|
| Ліва кнопка (натиснути і відпустити) | Намалювати лінію на кадрі |
| Колесо миші вгору | Збільшити зум |
| Колесо миші вниз | Зменшити зум |

### Трекбари (у вікні програми)
| Трекбар | Дія |
|---------|-----|
| Blur | Регулює силу розмиття в режимі Blur |
| Brightness | Регулює яскравість кадру |
