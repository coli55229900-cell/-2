# Camera App — Лабораторна робота №7

## Опис
Розширення програми з лабораторної роботи №6.
Додано детекцію облич в реальному часі на базі нейромережі ResNet-10
через OpenCV DNN модуль. Детекція винесена в окремий фоновий потік
щоб відео залишалось плавним без просадок FPS.

---

## Вимоги до системи
- **ОС**: Ubuntu 20.04 / 22.04
- **Залізо**: веб-камера, мінімум 2 ГБ RAM
- **ПЗ**: cmake, g++, make, libopencv-dev, wget

---

## Встановлення та запуск

### 1. Клонувати репозиторій
git clone <посилання на репозиторій>
cd lab7

### 2. Встановити залежності та завантажити нейромережу
./preinstall.sh

### 3. Зібрати проєкт
./build.sh

### 4. Запустити програму
./run.sh

### 5. Натиснути клавішу F для увімкнення детекції облич

---

## Архітектура

| Клас | Файли | Відповідає за |
|------|-------|---------------|
| CameraProvider | CameraProvider.hpp / .cpp | Захоплення кадрів з камери |
| KeyProcessor | KeyProcessor.hpp / .cpp | Обробка клавіатури, режим FACE |
| FrameProcessor | FrameProcessor.hpp / .cpp | Фільтри зображень |
| MouseHandler | MouseHandler.hpp / .cpp | Обробка миші |
| Display | Display.hpp / .cpp | Відображення кадру |
| FaceDetector | FaceDetector.hpp / .cpp | Детекція облич у фоновому потоці |

---

## Багатопотоковість

- **Головний потік**: читає камеру → малює рамки → imshow (30+ FPS)
- **Фоновий потік**: отримує кадр → запускає нейромережу → зберігає результат
- **std::mutex**: захищає спільні дані між потоками
- **std::atomic**: безпечна зупинка фонового потоку

Відео не гальмує — рамка оновлюється асинхронно.

---

| Клавіша | Дія |
|---------|-----|
